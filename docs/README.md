# GoGet Documentation

This folder contains product, technical, and execution docs for the GoGet concierge delivery app.

## Documents

- `PRODUCT_REQUIREMENTS.md` — problem definition, users, MVP scope, and success metrics.
- `SYSTEM_ARCHITECTURE.md` — system boundaries, services, data model, and operational constraints.
- `API_CONTRACTS.md` — HTTP endpoint contracts for clients, internal flows, and webhooks.
- `IMPLEMENTATION_ROADMAP.md` — phased plan from current build state to launch readiness.

## Reading order

1. `PRODUCT_REQUIREMENTS.md`
2. `SYSTEM_ARCHITECTURE.md`
3. `API_CONTRACTS.md`
